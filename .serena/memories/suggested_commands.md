# Suggested Commands

## Development
- `npm run dev` тАФ Start Next.js dev server
- `npx convex dev` тАФ Start Convex dev server (backend, real-time sync)

## Building
- `npm run build` тАФ Production build

## Testing
- `npm test` тАФ Run Vitest in watch mode
- `npm run test:run` тАФ Single test run
- `npm run test:coverage` тАФ Test run with coverage report

## Linting & Formatting
- `npm run lint` тАФ ESLint check
- `npm run format` тАФ Format all files with Prettier
- `npm run format:check` тАФ Check formatting without writing

## Production
- `npm run start` тАФ Start production server (after build)

## System Utilities (macOS / Darwin)
- `git` тАФ Version control
- `ls`, `find`, `grep` тАФ File system navigation and search
- `cat`, `head`, `tail` тАФ File content viewing
